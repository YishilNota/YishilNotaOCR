# YishilNotaOCR

**YishilNotaOCR** is a specialized desktop utility designed for processing PDF documents, performing Optical Character Recognition (OCR), and generating E-books (EPUB). It aims to provide a lightweight, "green" (portable), and efficient workflow for digital library enthusiasts.

## üåü Key Features

- **Advanced OCR Engine**: Convert scanned PDF pages or images into editable text (TXT) with support for complex layouts.
- **E-book Creator (EPUB)**: Convert processed text into professional EPUB files with native **Right-to-Left (RTL)** support for Uyghur script.

## üõ† Tech Stack

This application is built using a high-performance, modern .NET stack:

- **Framework**: `.NET 8 / WPF` (Windows Desktop)
- **Browser Engine**: `Microsoft.Web.WebView2` (Chromium-based)
- **PDF & Image Logic**: `Magick.NET` (ImageMagick wrapper for .NET)
- **Rendering Engine**: `Ghostscript 10.x` (PostScript/PDF interpreter)
- **UI Framework**: `Material Design in XAML Toolkit`
